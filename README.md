## API Endpoints

| Method | Endpoint  | Description         |
|--------|-----------|---------------------|
| GET    | `/`       | API info            |
| GET    | `/health` | Health check status |

## Getting Started

```bash
# Run the server
npm start

# Development mode (auto-reload)
npm run dev
```

Server runs on `http://localhost:3000` by default.
